Elementary steps of Ammonia dissociation

Ammonia adsorption and desorption step

  • NH3 (gas) + * → NH3* (R1), k1

NH3 dissociation is done in 3 elementary steps

  • NH3* + * ⇄ NH2* + H* (R2), kf2, kb2

  • NH2* + * ⇄ NH* + H* (R3), kf3, kb3

  • NH* + * ⇄ N* + H* (R4), kf4, kb4

N2 formation is done in 3 elementary steps

  • 2 N* ⇄ N-N** (R5), kf5, kb5

  • N-N** → *N2 + * (R6), k6

  • N2* → N2 (gas) + * (R7), k7

H2 formation is done in one elementary step

  • 2 H* → H2 (gas) + 2* (R8), k8
